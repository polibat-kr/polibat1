## 📋 Phase 2 개요

### 목표
- 댓글 시스템 완성 (CRUD, 대댓글, 좋아요/신고)
- 투표 시스템 완성 (생성, 참여, 결과 집계, 중복 방지)
- 불편/제안 접수 시스템 (관리자 답변, 상태 관리)
- 파일 업로드 (S3 연동, 이미지 최적화)
- 이메일 발송 (SES, 템플릿)
- AI MVP 기능 (3줄 요약, 감정 분석 온도계)
- Frontend Website API 연동 (page-manager.js, login.js)

### 산출물
- 댓글 API 5개 엔드포인트
- 투표 API 6개 엔드포인트
